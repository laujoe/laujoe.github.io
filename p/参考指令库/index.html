观点

这篇文章最让我震惊的是 “Aha Moment” 的出现。

模型在长时间的训练后，自动学会了通过"反思"和"验证"来解决问题。这对于我们做个人知识库也有启发：

  1. 少即是多：不要单纯堆砌数据（SFT），要注重逻辑的自我推演（RL）。
  2. 验证机制：写笔记时，要像模型一样，多问自己"为什么"，进行自我验证。

核心概念

Local-First

Local-First 是一种软件开发理念，强调数据的所有权属于用户，优先存储在本地设备上，云端仅作为同步和备份的辅助。

emoji

Emoji can be enabled in a Hugo project in a number of ways.

The emojify function can be called directly in templates or Inline Shortcodes .

To enable emoji globally, set enableEmoji to true in your site’s configuration and then you can type emoji shorthand codes directly in content files; e.g.

🙈 :see_no_evil: 🙉 :hear_no_evil: 🙊 :speak_no_evil:
